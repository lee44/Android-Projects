Pushing:
To initialize a new local repo
git init 

To add files to local repo
git add . 

To add a message to a commit
git commit -m "Your Message"

Sets up the remote repo you want to push to
git remote add origin <https or ssh>

To push to remote repo. origin is the remote name while master is the branch to push on
git push origin master 

Setting up SSH Key
https://help.github.com/articles/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent/

After key is generated and set in github, need to set the remote url(https or ssh). The remote url is the link to your remote repo on github.
https://help.github.com/articles/changing-a-remote-s-url/

git pull origin master --allow-unrelated-histories
=======
git remote set-url origin <ssh url>
